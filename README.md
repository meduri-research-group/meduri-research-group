# Meduri Research Group

This is a simple, GitHub-Pages-ready research group website (static HTML/CSS).

## Quick start (local)
Just open `index.html` in your browser.

## Deploy on GitHub Pages
1. Create a GitHub repo (e.g., `meduri-research-group`).
2. Upload all files from this folder to the repo root.
3. In GitHub: **Settings â†’ Pages**
   - Source: Deploy from a branch
   - Branch: `main`
   - Folder: `/ (root)`
4. Your site will be live at:
   `https://<username>.github.io/<repo>/`

## Customize
- Replace the placeholder portrait: `assets/img/avatar.svg` with a real photo (keep name same or update `index.html`).
- Replace the hero background: `assets/img/hero.svg`
- Edit page content:
  - `about.html`, `research.html`, `publications.html`, `group.html`, `contact.html`
